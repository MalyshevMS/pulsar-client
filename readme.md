# Pulsar
Это открытый клиент нашего мессенджера [Pulsar](http://pulsar-chat.ru)\
далее перечисленны возможности нашего мессенджера, а так же информация для разработчиков

# Возможности

 - ### Быстродействие
Вы можете скачать и сразу же пользоваться нашим мессенджером. Достаточно пройти простую регистрацию
 - ### Публичность
Все каналы и пользователи доступны для каждого.
 - ### Анонимность
Не смотря на открытость в переписки ваши личные данные данные никто не может узнать, пока вы сами их не укажите
 - ### Кросс-платформеннность
Наш мессенджер доступен на Windows 8.1+, [Linux](#поддерживаемые-дистрибутивы-linux), а так же [веб-версия](http://pulsar-chat.ru), доступная на любом современном браузере.
 - ### Шифрование (в процессе)
В следущей версии Pulsar (v0.1.2) уже появится современное самописное сквозное шифрование (на базе RSA)

### Форкинг и API

Вы можете самостоятельно создать свой клиент нашего мессенджера, клонировав этот репозиторий.\
***!!! Для отправки, получения и обработки сообщений рекомендуется использовать встроенный API (src/API/PulsarAPI.hpp) !!!***\
#### Формат сообщений:
```json
{
    "type": "message",
    "time": "[integer] current time (seconds since epoch)",
    "src":  "[string] this client username",
    "dst":  "[string] destination (channel or user)",
    "msg":  "[string] your message text"
}
```
#### Формат запроса на сервер
Если один аргумент: `!команда аргумент`\
Если больше: `!команда ["аргумент1", "аргумент2", ...]`
Запросы базы данных:
1) действия с контактами: `!db contact add/rem/get аргумент`
2) информация о пользователе: `!db user аргумент`
3) информация о канале: `!db channel аргумент`
#### Формат ответа от сервера:
`[+/-]команда_из_запроса ключевое_слово`
